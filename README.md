# JSON.LD

An implementation of the [JSON-LD](https://en.wikipedia.org/wiki/JSON-LD) standard.


## Installation

The package can be installed by adding `json_ld` to your list of dependencies in `mix.exs`:

    ```elixir
    def deps do
      [
        {:json_ld, "~> 0.1.0"}
      ]
    end
    ```

Then, update your dependencies:

```sh-session
$ mix deps.get
```
